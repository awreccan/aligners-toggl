Dev setup (already done for you)

Toggl’s API can’t be called from a browser, so a tiny stateless Google Apps Script relay sits in between. The dev deploys it once; it holds no tokens. Paste its /exec URL here only if it isn’t already built into the app.

Your token is stored only on this device and sent through the relay to Toggl. Your wear data lives in your Toggl — never anyone else’s account. Revoke the token anytime in Toggl’s profile settings.
