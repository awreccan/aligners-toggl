Dev setup (already done for you)

Toggl’s API can’t be called from a browser, so a tiny stateless Google Apps Script relay sits in between. The dev deploys it once; it holds no tokens. Paste its /exec URL here only if it isn’t already built into the app.

Your token is stored only on this device and sent through the relay to Toggl. Your wear data lives in your Toggl — never anyone else’s account. Revoke the token anytime in Toggl’s profile settings.

Set up Siri

Say “Hey Siri, Aligners off” when you eat and “Aligners on” when you’re done — hands-free, phone locked — plus lock-screen reminders to put them back in. Both halves matter: the voice logs your time, the reminders make sure they go back in. Tap each step to open it; do them in order.

First connect the app to your Toggl (the Connect tab above). Then come back — the links below fill in with your own token automatically.

0Make the reminders list

Open the Reminders app (white icon with coloured dots). Tap Add List (bottom-right), name it exactly Aligners (capital A), pick any colour, tap Done.

Why reminders, not a plain alert? iPhone reminders ring on your lock screen even on silent — exactly what you want at the dinner table.

1Build the “Aligners Off” shortcut

Open Shortcuts, tap ＋ (top-right), tap the name at the top and rename it exactly Aligners Off. The exact name is how Siri knows what to run.

Action 2 — the 45-min nudge. Add Adjust Date: tap the blue Date → Current Date, keep it on Add, set 45 Minutes. Then add Add New Reminder: text Put your aligners back in 🦷, tap Show More, List = Aligners, Alert = the Adjusted Date chip.

Action 3 — the 90-min nudge. Same pair again: Adjust Date (Current Date, Add, 90 Minutes), then Add New Reminder Aligners out 90 min — wrap it up 🦷, List Aligners, Alert = that new Adjusted Date.

+45 min “Put your aligners back in 🦷”
+90 min “Aligners out 90 min — wrap it up 🦷”

Nudges count from when you take them out — a friendly “you’ve been out a while”, not a countdown. The real out-budget lives in the ring on the main screen.

3Make Siri run them hands-free

Open each shortcut’s details (the ⓘ button) and turn OFF “Ask Before Running.” That lets Siri run them with the phone locked.
